# nvim
neovim config for krakenflex

# Install

This repo should be cloned to `~/.config/nvim`:

```
mkdir -p ~/.config
mv ~/.config/nvim ~/.config/_nvim 2>/dev/null
cd ~/.config
git clone git@bitbucket.org:ayubmalik-coachusa/nvim.git
```
