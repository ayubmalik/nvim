# README

## Neovim Reset

Danger! This deletes all neovim config files.


```shell
rm -rf ~/.local/share/nvim
rm -rf ~/.local/state/nvim
rm -rf ~/.cache/nvim
```

## Tools Dependencies

```
sudo apt install make gcc ripgrep unzip git xclip
```
